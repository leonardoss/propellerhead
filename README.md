# Propellerhead

Application created to Developer Technical Test of Propellerhead
##### Application created based on
- [Create React App][df-create-react-app]
- [MongoDB][df-mongodb]
- [MaterialUI][df-material-ui]
- [Axios][df-axios]

## How to run
##### Versions
* Node - 8.9.4

Install dependencies
```
$ npm install
```

Run the project
```
$ npm start
```
